# Product Roadmap

MenuBarHub's vision: become the **unified control center** for the macOS menu bar.

## Phase 1 — Foundation (current)

The basics: detect, display, and interact with running apps.

- [x] Grid panel showing all running apps (menu bar + Dock)
- [x] Smart filtering — exclude system processes, detect real menu bar apps
- [x] Click to activate Dock apps / trigger menu bar app dropdowns
- [x] Search by app name
- [x] Global hotkey `⌥ Space` to toggle the panel
- [x] Right-click context menu (Switch to / Quit)
- [x] Active app indicator (green dot)
- [x] Auto-refresh every 2 seconds

## Phase 2 — Organization

Let users take control of how apps are arranged.

- [ ] **Drag-and-drop reordering** — Rearrange icons within the grid, persist order across launches
- [ ] **Custom groups** — Create named groups (e.g. "Work", "Network", "System") and assign apps to them
- [ ] **Pinned apps** — Pin frequently used apps to the top of the panel
- [ ] **Hide apps** — Hide apps you never want to see (with an "unhide" option)

## Phase 3 — Quick Actions

Interact with menu bar apps without opening their full menus.

- [ ] **Quick actions panel** — One-click actions like "Connect VPN", "Toggle Do Not Disturb", "Pause Time Tracking"
- [ ] **Menu item preview** — Hover to see a preview of a menu bar app's dropdown content
- [ ] **Keyboard navigation** — Arrow keys to navigate the grid, Enter to activate

## Phase 4 — Power User Features

For users who want maximum efficiency.

- [ ] **Command palette** — `⌘K` style fuzzy search across all apps and actions
- [ ] **Custom hotkeys** — Assign keyboard shortcuts to individual apps
- [ ] **Status dashboard** — Aggregate status info from menu bar apps (e.g. VPN connected, battery level, upload speed)
- [ ] **Profiles** — Switch between sets of running menu bar apps (e.g. "Work mode" vs "Personal mode")

## Phase 5 — Polish

- [ ] **Themes** — Light, dark, and custom color schemes
- [ ] **Icon customization** — Replace app icons with custom ones
- [ ] **Homebrew Cask** distribution
- [ ] **Auto-update** via Sparkle
- [ ] **Localization** — English, Chinese, Japanese
